BGM files
=========

Place loopable BGM files here. Each track can be `.mp3`, `.ogg`, or both.

- `opening.mp3` / `opening.ogg`: title, opening mood, ending
- `town.mp3` / `town.ogg`: street field
- `club.mp3` / `club.ogg`: club interior field
- `battle_guest.mp3` / `battle_guest.ogg`: customer battle
- `battle_cabajo.mp3` / `battle_cabajo.ogg`: cabajo battle

The game skips BGM playback when a file is not available.
